# Designing-a-System-for-Customer-Request-Prioritization-Based-on-AI-Analysis-of-Feedback


## 📋 Описание

Цифровой прототип системы интеллектуальной приоритезации обращений клиентов на основе AI-анализа текстовых отзывов. Решение автоматически оценивает срочность обращений, ранжирует их по приоритету и генерирует черновики ответов с использованием методов NLP и компактных языковых моделей.

## 🎯 Цель проекта

Разработка MVP модульной интерпретируемой системы, которая принимает на вход текст отзыва клиента в электронной коммерции.Система формирует комплексную оценку для автоматической маршрутизации в службу поддержки, генерирует черновик персонализированного ответа, а также предоставляет оператору объяснение принятого решения через механизм объяснимого искусственного интеллекта (XAI) с визуальной подсветкой ключевых фраз и декомпозицией итогового балла.

## 📄 Описание признаков

| Столбец | Тип данных | Описание | Пример |
|---------|-----------|----------|--------|
| `category` | `string` | Категория товара | `"Ноутбуки"`, `"Смартфоны"` |
| `product_name` | `string` | Полное наименование товара | `"Ноутбук Huawei MateBook D 16..."` |
| `advantages` | `string` | Достоинства из отзыва | `"Шустрый, лёгкий, хороший экран"` |
| `disadvantages` | `string` | Недостатки из отзыва | `"Нет драйверов, греется"` |
| `comments` | `string` | Свободный текст отзыва | `"Не работает после обновления, срочно верните деньги!"` |
